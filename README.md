# Proyecto Escuela 2026

Este proyecto es una aplicación web para la gestión escolar.

## Tecnologías utilizadas
* **Next.js**: Framework de React.
* **Prisma**: ORM para la base de datos.
* **PostgreSQL/SQLite**: Motor de base de datos.

## Cómo empezar
1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Configurar el archivo `.env` con tu DATABASE_URL.

3. Ejecutar Prisma:
   ```bash
   npx prisma generate
   ```

4. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
